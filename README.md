# Portfolio Personal Profesional

Portfolio estático de Lidia Jiménez Soriano preparado para GitHub Pages, con estética dashboard pastel y contenido orientado a desarrollo RPA, UiPath, automatización y soluciones software.

## Archivos

- `index.html`: estructura principal del portfolio.
- `styles.css`: variables, layout responsive, tarjetas, animaciones y estilo visual.
- `script.js`: menú móvil, sección activa y animaciones de entrada.
- `README.md`: instrucciones del proyecto.

## Personalización rápida

1. Actualiza los enlaces de GitHub y LinkedIn en `index.html` si cambian tus perfiles.
2. Cambia los enlaces de proyectos y demos cuando tengas repositorios o despliegues publicados.
3. Edita los textos de experiencia, formación y proyectos cuando quieras actualizar el CV.
4. Sustituye `assets/cv.pdf` o `assets/perfil.jpeg` cuando quieras renovar el CV o la foto de perfil.

## Publicar en GitHub Pages

1. Sube estos archivos a un repositorio de GitHub.
2. En GitHub, entra en `Settings > Pages`.
3. En `Build and deployment`, selecciona `Deploy from a branch`.
4. Elige la rama `main` y la carpeta `/root`.
5. Guarda los cambios y espera a que GitHub genere la URL.

## Integrar formulario real

El formulario usa `mailto` como comportamiento básico. Puedes reemplazar el atributo `action` por el endpoint de Formspree, EmailJS u otro servicio.
